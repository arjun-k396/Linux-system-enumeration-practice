## System Monitoring Commands

This folder contains screenshots of Linux system monitoring commands executed in Kali Linux.

Commands covered:

- uptime – Show system running time and load average
- top – Display running processes and CPU usage
- htop – Interactive process viewer
- free -h – Display memory usage
- vmstat – Show system performance statistics
- iostat – Display CPU and disk I/O statistics
- mpstat – Show CPU usage per processor

These commands are commonly used for monitoring system performance and resource usage in Linux environments.
