# Linux System Enumeration Practice Project

🎓 Cybersecurity Student | Ethical Hacking Learner  
💻 Practicing Linux system enumeration and monitoring commands used in cybersecurity and ethical hacking labs

## Topics Covered

### 1️⃣ User & System Information Commands
whoami – Display the current logged-in user  
id – Show user identity and group information  
uname -a – Display system and kernel information  
hostname – Show the system hostname  

📸 Screenshots: screenshots/system-info/

---

### 2️⃣ System Monitoring Commands
top – Display running processes and system resource usage  
htop – Interactive process viewer  
uptime – Show system running time and load average  

📸 Screenshots: screenshots/system-monitoring/

---

### 3️⃣ Process Management Commands
ps aux – Display all running processes  
ps aux | grep process – Search for a specific running process  

📸 Screenshots: screenshots/process/

---

### 4️⃣ Disk Information Commands
df -h – Display disk space usage  
du -h – Show directory space usage  
lsblk – Display information about storage devices  

📸 Screenshots: screenshots/disk/

---

### 5️⃣ System History & Logs
history – Display previously executed commands  
last – Show login history of users  

📸 Screenshots: screenshots/logs/

---
